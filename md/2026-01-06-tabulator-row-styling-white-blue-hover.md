# Tabulator Row Styling - White Rows with Blue Hover

**Date:** 2026-01-06

## Summary

Updated Tabulator grid styling to use white rows with light blue hover effect.

## Changes Made

### Light Mode
- Removed alternating grey background on even rows (`#fafafa`)
- Changed row hover color from grey (`#f9fafb`) to light blue (`#eff6ff`)

### Dark Mode
- Removed alternating darker background on even rows (`#111827`)
- Changed row hover color from grey (`#374151`) to dark blue (`#1e3a5f`)
